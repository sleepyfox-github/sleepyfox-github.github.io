leetcode 203.移除链表元素

发表于2024-02-06|更新于2024-05-16|数据结构与算法链表
|字数总计:343|阅读时长:1分钟|阅读量:

问题描述

给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。

示例 1：

输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]

示例 2：

输入：head = [], val = 1
输出：[]

示例 3：

输入：head = [7,7,7,7], val = 7
输出：[]

提示：

  • 列表中的节点数目在范围 [0, 104] 内
  • 1 <= Node.val <= 50
  • 0 <= val <= 50

核心思路

删除：暂存指针，移动指针，释放内存

实现要点

借助虚拟头节点实现原始头节点和普通节点删除操作的统一性。

不使用虚拟头节点也可以，只是需要先把头节点删干净，使得头节点不是需要删除的对象，再判断后续普通节点是否需要删除。

code

  • 时间复杂度：O(n)O(n)O(n)
  • 空间复杂度：O(1)O(1)O(1)
